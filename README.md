# Data Structures and Algorithms in C

## Introduction

This repository focuses on implementations of data structures and alogorithms in the C programming language.  The C programming language is a great tool to implement algorithms because it teaches the user implementation details while forcing them to manually manage memory.  The sections will focus on explaining the functions for each of the data structures and algorithms.  

In order to study from this repository, please go from the first data structure to the last.  We will cover data structures that build on top of one another, so for example if you skip the linked list section and go to the hash map, then you may not understand our implementation.
